# Sistema de Gerenciamento de Restaurante 🍴

### Qual a proposta deste sistema?
- Temos como objetivo montar um sistema completo de um restaurante.

  ## Qual foi a nossa ideia?

  - Pensamos em trazer "algo" não apenas para realizar pedidos, mas para aplicar conceitos e ideias que vão trazer muito mais que um prato.
  - Queremos trazer tanto a parte web quanto a parte operacional dos sistemas.
